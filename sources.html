Data Sources

Links and descriptions for internet-hosted data of possible use when demonstrating data science projects.

Collections

  • sklearn - Examples for pythonic data science. skl.fetch_* to download, data.load_* to load to memory. Stored as sklearn input vectors. Includes downloading of various image and vector-based data ideal for learning.
    URL
  • Seaborn - Used in seaborn examples. Available simply as sns.load_dataset(name); list with sns.get_dataset_names(). The URL describes the origin of the datasets.
    URL
  • R - Since R is common in data science/machine learning, these are commonly seen. Many can be loaded automatically with statsmodels.
    URL
  • Data shader - Some examples of using some of the data sources on this page, with some further details of what processing might be required for each.
    URL
  • Other - miscellaneous collection of all sorts of data that might be useful.
    URL

General

  • Faces - See this example for how to use these images, common for face detection problems. Description
    URL
  • GDELT
    URL
  • Gutenberg - Full text of books out of copyright, one file per book (each with a short notice at the start). They are simply numbered, going up to the 10,000s; the example is Pride and Prejudice.
    URL
  • Flickr - Photosets. You will need to register and obtain an API key. The following code shows a possible use.
    photoset = requests.get( url ).json()
    photo = photoset['photoset']['photo'][0]
    flickr_url = "https://c2.staticflickr.com/{farm}/{server}/{id}_{secret}_n.jpg".format(**photo)
    import skimage.io
    im = skimage.io.imread(flickr_url)
    URL
  • NYC Taxi - JSON lines describing trips in NYC in 2012. In the example URL, the integer is in the range 0-12.
    URL
  • Wikipedia - Full dump of all articles as of September 2015.
    URL

Streaming

  • Open weather - Hourly (or less) free weather information from around the world. Register for an API id/key pair and read the description here.
    URL
  • Met office - Many national weather services also provide measurements and predictions. This reference is for the UK Met-Office.
    URL
  • Weather APIs - Collection of weather-providing APIs on the web, with some descrition of each.
    URL
  • Nextbus transit - GPS bus locations and per-stop predictions in real time (~1min updates) for several major North Americal cities.
    URL
  • USA government - Stream of URLs as they are fed through a minimizer. Other stream and REST end-points are also described.
    URL
